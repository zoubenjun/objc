# OC知识记录

该项目主要是整理记录一些OC相关的知识、代码。目的是为了方便以后复习回顾，如果看到这个项目的你能够有所收获那就更好了。
项目中可能存在个人理解错误的知识，若是发现，请指教。

- objc底层数据结构
- 内存管理
- Catagory（分类）
- RunLoop
- Runtime（包括源码-781）
- 多线程（NSThread、GCD、NSOperation）
- Block
- 网络
- 动画
- 音视频
- 性能优化
- 常用第三方库的学习
- 设计模式和架构
- 数据结构和算法
- iOS逆向
